Python-spring2016-codefirst

Lesson material for University of Southampton's CodeFirst Python Course: February 1 - March 21, 2016

View on GitHub Download .zip Download .tar.gz

Who: The course is aimed at female students and researchers who would like to learn good programming techniques and tools to develop their software more effectively and productively.

Requirements: Participants must bring a laptop with a few specific software packages installed (listed here).

Contact: This workshop is being organised by the University of Southampton and CodeFirst:Girls . Please mail {info@codefirstgirls.org.uk} for more information.

What should I bring with me?

You must bring your own laptop! Before attending the workshop you need to install some useful and free software. For more information, see the software prerequisites.
